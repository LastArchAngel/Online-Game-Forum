# Online-Game-Forum
A project so that I can practive .NET MVC.
Basic gaming based web-forum that allows users to talk about games and trade/sell games.

I will be using
  * Microsoft Visual Studio 2017
  * ASP.NET 4.7
  * Entity Framework 6.2
  * Javascript/JQuery
  * Bootstrap/Bootbox
  * Paypal API
  * tawk.to API
